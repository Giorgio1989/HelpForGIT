# HelpForGIT
---
## Команды для bash
cd - смена директории
mkdir - создать директорию
touch - создать файл
rm - удалить
ls - показать список файлов
---
## Команды для GIT
init - инициализировать
add - добавить файл (. - добавить всю папку)
status - узнать статус файлов - готовность к коммиту
commit - закоммитить (-m - дать комментарий)
